--------------------------- Slicer ---------------------------

OPIS SLICERA NA ENVATO:

Jeśli szukasz rozwiązania, ktore zaimplementujesz na momencie to trafiłeś idealnie!
Zanim kupisz przekonaj się do czego jest zdolny używając generatora na stronie slideshowslcer.com!
- Slicer to prosty, lekki, customizowalny tool do generowania sliderow na strony internetowe
- Slicer uses simple algorythms to manage attributes of the slices that each slider consists of.
- Slidery tworzone przy jego pomocy mogą przyjmować zdefiniowane przez użytkownika rozmiary, przez co mogą być używane na responsywnych stronach internetowych
- Slicer jest dziecinnie prosty do instalacji na dowlnej stronie napisanej w htmlu. Załączony z paczką przykład jest krotki i przejżysty dzięki czemu bez trudu podepniesz slicer pod swoją stronę, jedynie kopiując fragmenty kodu
- Kompletny tutorial i dokumentacja znajdują się na slideshowslider.com
- Slicer jest narzędziem w ktorym istnieje praktycznie nieograniczona liczba animacji jakie można wygenerować. Tworzenie animacji jest trywialne dzięki generatorowi (slideshowslider.com/#generator) za pomocą ktorego można manipulowac parametrami slicera a następnie obejrzeć efekty i wyeksportować kod -> znajomość js jest niepotrzebna. Przykłady można obejrzeć w zakładce 'examples'
- Slicer bazuje na velocity js dzięki czemu animacje są płynne a co za tym idzie - przyjemne dla oka
- Dzięki slicerowi na swoim sliderze możesz umieścić rownież wiadomości (opisy slajdow). Je rownież możesz customizować wpływając na ich wygląd i sposob pojawiania się.

FEATURES NA STRONKE:

Slicer IS:
- easy to use: All what you need to use Slicer is attach appropriate js and css files, and write or copy a piece of html and js code. You don't have to have knowledge of programing. Try to set your custom parameters in generator[link] and generate code. It takes 5-15 minutes to prepare full slider to any web page.
[time icon]

- beatifull: Slicer is based on velocity engine which guarantee smoother animations than clear jQuery animations and that's why Slicer's transitions are beatifull and pleasing for eyes.
[eye-open]

- very customizable: Slicer is not only the set of ready-for-use defined sliders - it is tool which can generete a wide variety of sliders. You can customize transitions in many ways by setting appropriate parameters. With Slicer you can decide how your animation will look like without any constraints.
[pencil]

- funkcjonalny: Beside possibility of customizing transitions between slides you can also add description to each slide. Moreover you can adjust way of its appearing as well as position and appearance parameters. 
[wrench]

- responsywny (link do responsinatora): You can define size of your slider and that's why you can adjust it to any screen resolution by setting other size depending on device on which page is viewed.
[phone]

- light weight: Main file of Slicer (slider.js) weights only ~30KB which means that it will not slow down your page
[scale]



NA ENVATO DODAĆ:

What is Slicer?
Slicer is simple, easy to use and customizable tool for creating sliders for web page. It uses simple algorythms to manage attributes of the slices that each slider consists of. 

Why Slicer?
If you are looking for solution which you can adjust to your page in few minutes you found ideal place. Before you buy Slicer you can see it in action by using generator placed in Slicer Home Page where you can try to set appropriate parameters, see predefined examples and, most importantly, generate code.
